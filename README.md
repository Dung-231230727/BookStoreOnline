# 📚 BookStore Online - Hệ thống Quản trị Nhà sách

Dự án quản lý nhà sách toàn diện bao gồm: Backend API, Tài liệu thiết kế và Database.

---

## 📂 Cấu trúc thư mục
- 📁 **`bookstoreonline/`**: Mã nguồn Backend (Java 21, Spring Boot 3.3).
- 📁 **`tailieu/`**: Danh sách 49 API, Kế hoạch 7 ngày, Phân chia nhiệm vụ [MB A-E].
- 📁 **`database/`**: Script khởi tạo SQL Server (`bookstore_db.sql`).

---

## 🛠 Tech Stack (Backend)
- Java 21, Spring Boot 3.3
- Spring Security 6 & JWT
- MS SQL Server, JPA/Hibernate
- Swagger / OpenAPI 3.0

---

## 🚀 Hướng dẫn nhanh cho Team
1. **Clone**: `git clone <url>`
2. **Database**: Chạy file SQL trong thư mục `database/` trên SQL Server.
3. **Config**: Cấu hình mật khẩu SA tại `bookstoreonline/src/main/resources/application.yml`.
4. **Code**: Xem phân công nhiệm vụ trong `tailieu/api.txt` và bắt đầu thực hiện theo chuẩn `ApiResponse`.
